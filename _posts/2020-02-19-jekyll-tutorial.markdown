* Create a New Ogranization on Github

* Create a New Repository in that organization named: **`organization_name.github.io`**

* Copy the Repository URL

* Open VSCode -> Press F1 -> Git: Clone -> paste repo URL -> Select Folder

* Open Terminal: ctrl + `

* Run `Jekyll new .`
